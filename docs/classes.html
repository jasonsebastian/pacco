class pacco.classes.PackageRegistry(name: str, client: pacco.classes.Client, allowed_settings: Optional[Dict[str, List[str]]] = None)¶

A PackageRegistry instance represent a package collection and client pair. This class shall depend on Client but not the other way around.

An instantiation of this class represents the existence or the intend to make exist of the package collection self.name in the client self.client.
